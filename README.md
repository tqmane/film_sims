# FilmSims — フィルムシミュレーター

**写真に、フィルムの息吹を。**  
フィルムストックやカメラブランドの色調を再現する写真編集アプリ。

---

## 特徴

| 機能 | 詳細 |
|---|---|
| 🎞 **フィルムシミュレーション** | 多彩なプリセットから好みの色調をワンタップで適用 |
| 🎚 **エフェクト強度** | 0〜100% の範囲でシミュレーションの適用量を細かく調整 |
| 🌾 **フィルムグレイン** | ノイズ量・粒子感をリアルタイムでコントロール |
| 🔍 **ピンチズーム** | 2 本指でプレビュー画像を直感的に拡大・縮小 |
| 🗂 **フィルター分類** | ブランド・ジャンル別に整理されたプリセットライブラリ |
| 💾 **高画質書き出し** | オリジナル解像度 + Exif 情報を保持したまま保存 |
| ⚡ **GPU 高速処理** | OpenGL ES 3.0 による GPU アクセラレーションで快適な編集体験 |

---

## 技術スタック

- **言語:** Kotlin
- **UI:** Jetpack (Compose / View)
- **画像処理:** OpenGL ES 3.0
- **ターゲット SDK:** Android SDK 34
